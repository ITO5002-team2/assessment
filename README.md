# Vaccination Coverage Dashboard

## Idea

Vaccination Coverage and Awareness Dashboard for clinicians, community health workers, and parents/guardians in Australia.

The project supports **UN Sustainable Development Goal 3: Good Health and Well-Being** by making childhood immunisation data easier to understand and use.

Australia's official data shows that childhood vaccination coverage is **below ideal levels in some age groups and regions**, making this a realistic and relevant data-driven project.

## Project Goal

The goal of this project is to build a clear, interactive dashboard that:

- displays vaccination coverage by geographic area
- compares coverage levels across regions
- highlights areas with lower immunisation coverage
- provides contextual information to help users interpret regional variation
- helps clinicians quickly assess local vaccination coverage and risk patterns

## Data Sources

- **Australian Government – Childhood Immunisation Coverage**  
  https://www.health.gov.au/topics/immunisation/immunisation-data/childhood-immunisation-coverage

- **Australian Bureau of Statistics (ABS) – Regional Population by Age and Sex**  
  https://www.abs.gov.au/statistics/people/population/regional-population-age-and-sex/latest-release

## Key Stakeholders

- Primary Persona: Clinicians, doctors, and community health workers
- Secondary Persona: Parents/Guardians

## Running the Code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.

Run `npm run build` to create a production build.
