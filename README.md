## Idea
Vaccination Coverage and Awareness Dashboard for parents and carers of young children in Australia.  
The project supports **UN Sustainable Development Goal 3: Good Health and Well-Being** by making childhood immunisation data easier to understand and use.

Australia’s official data shows that childhood vaccination coverage is **below ideal levels in some age groups and regions**, making this a relevant and realistic data-driven project.

## Concerns / Problems (Why They Are Important)

Childhood vaccination is one of the **most important preventive health measures**. However, vaccination coverage is **not uniform across Australia**.

Some communities have **lower vaccination rates**, which increases vulnerability to preventable diseases. In addition, official health datasets can be **difficult for parents and carers to interpret**, creating a barrier to understanding local immunisation trends.

This makes the issue both a **public health concern** and an **information accessibility challenge**.

---

## Project Goal and Rationale

The goal of this project is to build a **simple, interactive dashboard** that:

- Displays vaccination coverage by geographic area  
- Compares coverage levels across regions  
- Highlights areas with lower immunisation coverage  
- Overlays regional child population data for context  

By transforming official public datasets into a **clear and accessible visual tool**, the dashboard will help families, communities, and health organisations better understand immunisation patterns and support informed decision-making.
